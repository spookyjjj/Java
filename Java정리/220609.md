  ```
  =========format 정리=========
  %5d : 5자리, 오른쪽 정렬  
  %05d : 5자리, 공란은 0으로 채움, 오른쪽 정렬  
  %-5d : 5자리, 왼쪽 정렬  
  %,d : 천의자리마다 쉼표
  -----------------------  
  %12s : 최소 12자리, 오른쪽 정렬 //        tete//
  %-12s : 최소 12자리, 왼쪽 정렬 
  %.2s : 최대 2자리 -> 넘어가면 짜름 //te//
  &12.2s : 공간은 12자리, 내용물은 2자리 //          te//  
  ----------------
  참조 https://blog.jiniworld.me/68
  ```
